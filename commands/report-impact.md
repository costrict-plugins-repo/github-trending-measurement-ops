# /report-impact
Build an evidence-based impact report.
## Steps
1. **Gather the numbers** — Pull KPI movement against baseline using `program-kpis` skill.
2. **Show the trend** — Visualise before/after using `health-dashboard` skill.
3. **Write the story** — Connect outcomes to value using `impact-reporting` skill.
## Output
An impact report: outcomes, evidence, honest attribution, cost, and a recommendation for what's next.
Consider following up with `/define-metrics` if any claim lacked a clean baseline.
